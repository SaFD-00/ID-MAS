# ID-MAS: 실험 결과

## 1. 개요

### 1.1 실험 설정

**ID-MAS (Instructional Design-based Multi-Agent Scaffolding)**는 교수설계 원리를 활용하여 교사-학생 모델 간 반복적 상호작용을 통해 고품질 SFT 학습 데이터를 자동 생성하는 프레임워크이다.

- **Student 모델**: Qwen3-0.6B, Qwen3-1.7B, Qwen3-4B, Qwen3-8B
- **Teacher 모델**: 동일 모델 계열의 상위 모델
- **도메인**: 수학적 추론, 논리적 추론, 상식 추론
- **평가 방식**: Accuracy (exact match), held-out test set 기준

### 1.2 벤치마크

| 도메인 | 학습 데이터 | In-domain 평가 | Out-of-domain 평가 |
|--------|-----------|---------------|-------------------|
| Math | GSM8K, MATH | GSM8K, MATH | SVAMP, ASDiv, MAWPS |
| Logical | ReClor | ReClor | ANLI-R2, ANLI-R3, BBH |
| Commonsense | ARC-C | ARC-C | StrategyQA, OpenBookQA |

### 1.3 SFT Case 분류

ID-MAS 스캐폴딩 과정에서 각 학습 샘플은 다음 세 가지 Case로 분류된다:

| Case | 설명 |
|------|------|
| **Case A**: Independent Performance Mastery | 학생이 1회차에 정답을 맞춘 경우 |
| **Case B**: Scaffolded & Coached Mastery | 교사 스캐폴딩을 통해 2~5회차에 성공한 경우 |
| **Case C**: Teacher Modeling Distillation | 최대 반복 후에도 실패하여 교사의 풀이를 사용한 경우 |

## 7. 주요 발견 및 논의

### 7.1 핵심 발견

1. **모델 크기가 ID-MAS 효과의 핵심 결정 요인이다.** 4B 이상 모델은 baseline에 근접한 성능을 유지하는 반면(|Δ| < 1.5 avg), Qwen3-0.6B는 모든 도메인에서 심각한 하락을 보인다(−7.07 ~ −19.36 avg).

2. **소형 모델의 catastrophic forgetting은 학습 성공과 무관하다.** Qwen3-0.6B는 Commonsense에서 100%, Math/GSM8K에서 99.93%의 학습 성공률을 달성했으나 가장 큰 평가 하락을 보여, SFT 과정에서 기존 지식을 유지할 모델 용량이 부족함을 시사한다.

3. **Out-of-domain 양의 전이는 드물지만 관찰 가능하다.** 유일한 유의미한 OOD 향상은 Qwen3-4B Commonsense의 StrategyQA(+5.53)이며, 적절한 크기의 모델에서 ID-MAS 스캐폴딩이 일반화를 향상시킬 수 있음을 시사한다.

4. **BBH에서 모델 크기별 성능 경향이 논리적 추론 능력을 잘 반영한다.** Baseline 기준 0.6B(47.20%) → 1.7B(68.13%) → 4B(90.04%)로 모델 크기에 비례하여 성능이 증가하며, ID-MAS 학습 후에도 소폭 하락(−0.62 ~ −3.60)에 그쳐 논리적 추론 능력이 대체로 보존됨을 보여준다.

5. **스캐폴딩(Case B)은 논리적 추론에서 가장 많이 활용된다.** Qwen3-4B가 66.7%의 Case B를 달성하여, 초기 시도에서 자주 실패하는 복잡한 추론 과제에서 반복적 교사-학생 상호작용이 특히 유의미함을 보여준다.

### 7.2 이상치 (Anomalies)

| 이슈 | 상세 |
|------|------|
| Math/MATH 4B: 59.3% Case C | MATH 데이터셋의 난이도와 4B 모델 용량 간 격차로 인해 교사 distillation에 크게 의존. |
| Commonsense 0.6B: 100% 성공, −19.36 평가 | 역설적 결과 — 완벽한 학습 성공이 평가 성능으로 이어지지 않으며, catastrophic forgetting을 강하게 시사. |

### 7.3 한계점

- **누락된 구성**: Math/MATH fine-tuning은 4B, 8B에서만 수행되었다.
- **단일 학습 실행**: 결과는 구성당 단일 학습 실행에 기반하며, 다중 시드 또는 신뢰 구간이 포함되지 않았다.
- **하이퍼파라미터 ablation 미실시**: SFT 하이퍼파라미터가 catastrophic forgetting에 미치는 영향이 탐구되지 않았다.

### 7.4 시사점

- **ID-MAS의 최소 모델 크기**: 결과는 **4B 파라미터**를 ID-MAS 적용의 실질적 최소 기준으로 제시하며, 그 이하에서는 catastrophic forgetting이 지배적이다.
- **도메인 난이도의 영향**: 수학적 추론이 가장 안정적인 ID-MAS 동작을 보이는 반면, 상식 추론은 모델 용량에 가장 민감하다.
- **스캐폴딩의 가치**: 논리적 추론에서의 높은 Case B 활용률은 ID-MAS 설계를 검증한다. 반복적 코칭이 어려운 문제에서 학생을 성공적으로 안내하지만, 이것이 항상 평가 성능 향상으로 이어지지는 않는다.
